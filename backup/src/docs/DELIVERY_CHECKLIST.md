# 最终交付检查清单

## 交付包

- [ ] 源代码版本、制品版本和 SHA-256 清单已记录。
- [ ] 数据库迁移、配置项说明和环境变量模板已交付。
- [ ] `USER_GUIDE.md`、`OPERATIONS_GUIDE.md` 和生产检查清单已评审。
- [ ] 自动化测试命令、环境、版本和结果已归档。
- [ ] 备份恢复演练记录、监控告警说明和故障升级联系人已交付。
- [ ] 已知限制、第三方依赖、许可证和遗留问题已书面说明。

## 客户金标准

- [ ] 客户提供最终样本和原始文件 SHA-256 清单。
- [ ] 客户签字确认盲测集版本、样本分布和隔离规则。
- [ ] 客户签字确认指标、阈值、容差、权重、严重缺陷和一票否决项。
- [ ] manifest 自校验及全部文件 SHA-256 校验通过。
- [ ] 使用冻结规则执行测试，JSON 报告和 Markdown 问题清单已归档。
- [ ] 报告明确区分开发合成预检与客户真实盲测结果。
- [ ] 客户业务审核人确认结果和遗留问题。

## UAT 与生产准备

- [ ] 客户按真实业务流程执行 UAT，失败项均有责任人和约定日期。
- [ ] UAT 报告中的解析、字段、数值、关联、表格和翻译结果已填写。
- [ ] 生产安全、配置、迁移、构建、监控、备份和恢复检查完成。
- [ ] 上线窗口、回退条件、支持期限和故障升级方式已确认。
