## 第 18-20 天开发预检

- 已提供 `gold_standard_manifest.py`，用于生成 manifest、自校验 manifest SHA-256，
  并逐项校验样本文件大小和 SHA-256。
- 已提供 `gold_standard_import.py`，用于在不写入业务数据库的前提下校验并封装
  `expected.json` 和 `actual.json`。
- 已提供 `gold_standard_evaluate.py`，计算解析完整率、字段准确率、数值容差准确率、
  关联准确率、表格结构准确率和翻译参考文本相似度，并输出 JSON 报告和 Markdown 问题清单。
- 纯函数测试和合成 fixture 仅验证评分逻辑、异常清单与命令行输入约束。
- 翻译相似度是对客户批准参考译文的可重复开发预检，不替代双语审阅人的质量盲评。
- 合成样本结果不构成客户真实准确率声明；客户样本、阈值、容差、权重和一票否决项尚待签字确认。
